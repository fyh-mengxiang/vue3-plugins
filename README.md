
## vue3-plugins目前主要是封装Vue3相关插件
## 仓库地址
仓库地址：https://github.com/yangxiaoguang/vue3-plugins
仓库地址：https://gitee.com/yangxiaoguang/vue3-plugins


### 1.vue3-event-bus 插件目前是开源的EventBus
（1).支持vue3项目安装
 npm install vue3-event-bus-plugin
（2).具体源代码可以查看目录vue3-event-bus

### 2.vue-img-lazy-directive 插件目前是开源的图片懒加载
（1).支持vue3项目安装
 npm install vue-img-lazy-directive
 （2).具体源代码可以查看目录vue-img-lazy-directive
