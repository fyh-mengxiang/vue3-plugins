vue3-plugins
Vue3 相关插件集合库

项目简介
vue3-plugins 是一个专注于 Vue3 生态系统的插件集合库，提供了一系列实用的 Vue3 插件，帮助开发者更高效地构建 Vue3 应用。

仓库地址
GitHub: https://github.com/yangxiaoguang/vue3-plugins
Gitee: https://gitee.com/yangxiaoguang/vue3-plugins
插件列表
1. vue3-event-bus
Vue3 事件总线插件，提供组件间通信能力。

安装
bash
npm install vue3-event-bus-plugin
使用方法
请参考 vue3-event-bus 目录下的源代码和示例。

功能特性
支持 Vue3 项目
提供事件发布/订阅机制
轻量级，易于集成
2. vue-img-lazy-directive
Vue3 图片懒加载指令插件，优化页面加载性能。

安装
bash
npm install vue-img-lazy-directive
使用方法
请参考 vue-img-lazy-directive 目录下的源代码和示例。

功能特性
支持 Vue3 项目
图片懒加载功能
提升页面加载性能
